## 核心用户场景

### 场景1: 新用户首次使用 (Onboarding)

#### 目标
- 快速理解产品价值
- 完成首个海报生成
- 建立成就感和信心

#### 流程

```
【第1步】访问首页
  ↓
用户看到:
  • Hero 部分: 产品标语 + 演示截图
  • CTA按钮: "立即开始" (突出显示)
  • 功能介绍: 4个核心优势
  • 评价/用户数据: 社会证明

用户行为:
  ✓ 有产品需求 → 点击 "立即开始"
  ✗ 没有产品需求 → 浏览功能介绍

【第2步】选择 "立即开始" / 登录或注册
  ↓
弹出注册对话框:
  • 邮箱输入 + 验证
  • 密码设置 (8+ 字符，需大小写数字)
  • 快速注册: Google/微信登录选项

用户行为:
  ✓ 填写信息 → 点击 "注册"
  ~ 也可选 "稍后注册" → 试用模式

【第3步】新用户欢迎流程
  ↓
欢迎屏幕展示:
  • 大标题: "欢迎来到智绘海报！"
  • 3个步骤说明 (图文结合)
    1️⃣ 输入内容 - 告诉我你的想法
    2️⃣ 选择风格 - 我们替你生成
    3️⃣ 导出分享 - 完美呈现！
  • CTA: "创建第一个海报" (蓝色按钮)

【第4步】新建项目 (首次简化版)
  ↓
简化的创建流程:
  • 预设尺寸快选: [1:1] [16:9] [9:16] [自定义]
  • 选择 1:1 (正方形) - 最常见
  • 显示即时预览

用户行为:
  ✓ 选择尺寸 → "下一步"

【第5步】内容输入 (引导式)
  ↓
分步输入，每步一个字段:

  第1个输入框:
    标签: "你要宣传的主题"
    placeholder: "例: 双11大促销"
    字数提示: 可获得更好的设计效果 (5-20字)

  用户输入后 → 自动保存 → 预览同步更新

  第2个输入框:
    标签: "补充信息 (可选)"
    placeholder: "例: 活动时间、优惠信息"

  第3个输入框:
    标签: "行动召唤 (可选)"
    placeholder: "例: 立即购买、了解更多"

  图片上传:
    "上传你的图片或Logo"
    支持拖拽上传 + 点击选择
    预设宽度比例提示

【第6步】选择风格
  ↓
展示 8 种风格卡片:
  • 每个卡片: 缩略图 + 风格名称 + 描述
  • 默认选中第一个 "简约商务" (推荐)
  • 悬停时显示大预览
  • 点击选中 (边框高亮)

新用户提示:
  "💡 不确定？试试'简约商务'，适合大多数场景"

【第7步】生成设计
  ↓
点击 "生成设计" 按钮

视觉反馈:
  • 按钮变为 loading 状态
  • 显示进度提示: "AI 正在设计..."
  • 进度条: 0% → 100%
  • 预计时间: "大约 8 秒"

等待期间:
  • 显示有趣的文案: "我们的 AI 正在思考..."
  • 或显示设计技巧: "好的海报需要..."

【第8步】预览变体
  ↓
生成完成，显示 3 个设计变体:
  • 并排显示 3 个缩略图
  • 下方有按钮: "选择这个设计"
  • 全屏预览选项: "查看大图"
  • 重新生成选项: "我要其他设计"

用户行为:
  ✓ 选择满意的设计 → "选择这个设计" → 进入编辑
  ~ 不满意 → "重新生成" → 回到第7步

【第9步】预览和微调
  ↓
进入编辑器:
  • 展示选中的设计在中央
  • 右侧显示可编辑属性
  • 提示: "可以微调标题、颜色等"

新用户可选操作:
  • 试试改标题 (点击文本双击编辑)
  • 试试改配色 (点击 "一键换色")
  • 跳过直接导出

【第10步】导出
  ↓
点击 "导出" 按钮

导出对话框:
  • PNG / HTML / PDF 三个标签
  • 默认 PNG (最常用)
  • 推荐分辨率: "中" (150DPI)
  • 显示文件预期大小

用户行为:
  ✓ 点击 "下载" → 文件下载开始

【第11步】完成 & 庆祝
  ↓
下载成功 → 弹出成功提示:

  ✓ 成功！海报已下载

  接下来可以:
  ☐ 创建更多海报
  ☐ 升级到 Pro 查看更多风格
  ☐ 分享到社交媒体

  [继续创建] [升级Pro] [关闭]

新用户成就徽章:
  "🎉 恭喜完成第一个海报设计！"
  在用户仪表板显示徽章
```

**优化要点**:
- ✅ 每步都有清晰的视觉反馈
- ✅ 输入字段使用 placeholder 示例
- ✅ 显示预期结果 (缩略图预览)
- ✅ 可以随时跳过、回退、重来
- ✅ 提供帮助提示，但不强制
- ✅ 成功时庆祝和激励

### 场景2: 运营人员批量生成海报

#### 目标
- 快速生成多个变体
- 批量调整和导出
- 提高工作效率

#### 流程

```
【第1步】从仪表板快速创建
  ↓
点击 "新建项目" → 快捷菜单:
  [选择模板] [从头开始] [上传CSV]

选择 "从头开始"

【第2步】填充内容并生成
  ↓
● 标题: "双11 限时特惠"
● 副标题: "全场5折起"
● 正文: "仅限 2025.11.1-11.11"
● CTA: "立即购买"
● 图片: 上传产品图 (product.png)

选择风格: "电商促销"
配色: 选择红黄配色

点击 "生成 5 个变体" (生成更多)

【第3步】快速对比变体
  ↓
5 个变体并排显示:
  [变体1] [变体2] [变体3] [变体4] [变体5]

每个变体下方:
  [选择] [预览] [编辑]

运营人员操作:
  ✓ 快速浏览，选择 2 个满意的设计

【第4步】批量微调
  ↓
对于选中的 2 个设计:

变体1 - 编辑:
  • 改标题字号 (从 48px → 56px)
  • 调整标题颜色 (深化一点)
  • 改 CTA 文案: "立即购买" → "查看优惠"

变体2 - 编辑:
  • 保持原样 (已满意)

【第5步】多格式导出
  ↓
对于变体1:
  • PNG 导出: 高分辨率
  • HTML 导出: 用于邮件推广

对于变体2:
  • PNG 导出

点击 "批量导出" 按钮
  ↓
系统自动生成:
  📦 下载文件
    ├─ 双11特惠_变体1.png
    ├─ 双11特惠_变体1.html
    └─ 双11特惠_变体2.png

【第6步】上传到营销平台
  ↓
运营直接:
  • PNG: 上传到微博、小红书
  • HTML: 粘贴到邮件模板

效率提升: 原需 1-2 天 → 现在 30 分钟 ✓
```

**关键交互**:
- 快速变体生成和对比
- 一次性修改多个设计
- 批量导出多种格式
- 保存设计以便复用

## 交互细节说明

### 1. 文本输入优化

#### 单行文本框 (Title/Subtitle)

```
【默认状态】
输入框: 高 36px, 左内边距 12px
placeholder: "输入标题，5-20字效果最佳"
右侧: 字数计数 (当前/最大)

【焦点状态】
border: 2px solid #0066CC
box-shadow: 0 0 0 4px rgba(0,102,204,0.1)
光标位置清晰

【输入中】
字数计数实时更新，文字在右侧预览同步显示

【错误状态】
输入内容为空或过短时:
  border: 2px solid #EF4444
  提示文案: "请至少输入 2 个字符"
  文字为红色

【完成状态】
输入完整后:
  右侧显示 ✓ 绿色勾
  transition: all 200ms ease
```

#### 多行文本框 (Body Text)

```
【功能】
• 支持换行 (Shift+Enter)
• 自动高度调整 (最小 80px, 最大 200px)
• 字数限制: 200 字
• 实时预览

【交互】
• 焦点时: 高亮边框 + 背景微变
• 过长提示: "还可输入 XX 字"
• 输入完成: ✓ 提示

【占位符文本】
"输入宣传文案，建议 20-100 字"
"❌ 避免超过 200 字，避免关键信息被遮挡"
```

## 错误处理流程

### 常见错误场景和处理

#### 1. 网络错误

```
【连接超时】
场景: 用户生成海报时网络断连

视觉呈现:
  界面显示: "⚠️ 连接超时"
  提示文案: "请检查网络连接后重试"
  按钮: [重试] [返回首页]

后台处理:
  • 自动保存草稿
  • 30秒后自动重连
  • 显示重连倒计时

【网络恢复】
  自动重试上次的操作
  显示 "连接已恢复" 的绿色提示 (2秒后消失)
  用户可继续操作
```

#### 2. 上传错误

```
【文件过大】
用户上传 > 20MB 的文件:

  ✗ 提示文案: "文件大小 25MB，超过 20MB 限制"
  解决方案: "请压缩后重新上传"
  建议: "使用在线压缩工具或本地编辑软件"
  按钮: [返回] [了解更多]

【格式不支持】
用户上传 .gif 或其他格式:

  ✗ 提示文案: "不支持 .gif 格式"
  支持的格式: "JPG, PNG"
  按钮: [选择其他文件]

【病毒扫描失败】
后台病毒扫描不通过:

  ✗ 提示文案: "文件安全检查失败"
  说明: "该文件可能包含风险内容"
  建议: "请确保文件来源可信"
  按钮: [更换文件] [联系支持]
```

#### 3. 生成失败

```
【超时失败】
场景: 生成海报超过 30 秒未完成

视觉呈现:
  Loading 状态显示: "⚠️ 处理时间较长"
  自动选项: [继续等待] [使用模板替代] [返回修改]

自动重试:
  • 后台自动重试一次
  • 如果再失败，显示备选方案

【算法错误】
场景: 排版引擎出现 bug

显示:
  ✗ "抱歉，生成失败"
  原因: "我们遇到一个临时问题"
  解决: "请稍后重试，我们已收到错误报告"
  按钮: [重新尝试] [使用上一版本] [反馈问题]

记录:
  • 自动上报错误日志
  • 包含用户内容 (用于复现)
  • 提供错误 ID: "ERROR_GEN_20251026_12345"
```

#### 4. 编辑冲突

```
【同步冲突】
场景: 用户在两个标签页同时编辑

检测:
  页面 A 进行了修改 → 本地保存
  页面 B 也做了修改 → 尝试上传

提示用户:
  页面 B 显示: "⚠️ 检测到其他地方的修改"

  选项:
  [ 使用这个版本 (放弃当前修改) ]
  [ 合并修改 (智能冲突合并) ]
  [ 查看差异 (diff 视图) ]

【离线编辑】
场景: 用户离线时编辑，恢复连接后冲突

处理:
  • 保留本地版本和云端版本
  • 显示: "✓ 本地修改已保存"
  • 提示: "恢复网络后会自动同步"
  • 给用户选择权: [云版本] [本地版本] [合并]
```

#### 5. 权限错误

```
【无权访问项目】
场景: 用户尝试访问别人的项目

显示:
  🔒 "无权访问"
  说明: "该项目不可见或已删除"
  选项: [返回首页] [查看分享链接示例]

【订阅过期】
场景: Pro 用户的功能在订阅过期后

显示:
  ⏰ "您的 Pro 订阅已过期"
  提示: "此功能仅 Pro 用户可用"
  选项: [升级续费] [查看免费功能] [继续使用免费版]

降级处理:
  • 新建项目时默认使用免费功能
  • 已有的 Pro 项目可继续查看但不可编辑
```

## 用户心智模型

### 用户心智模型构建

```
【用户心智要素】

1. 价值认知
   "我可以快速制作专业级海报"
   ↓
   期望: 时间短、质量好、易上手

2. 信任建立
   "这个工具值得信赖"
   ↓
   期望: 安全、稳定、有人支持

3. 效率感受
   "确实比之前快很多"
   ↓
   期望: 明确的进度、快速反馈

4. 成就感
   "我也能做出好看的设计"
   ↓
   期望: 高质量输出、可分享、获认可

5. 复用性认知
   "下次可以更快"
   ↓
   期望: 保存模板、批量生成
```

### 用户决策流程

```
【决策树】

用户进入产品
  ├─ 问题1: "这是什么？"
  │  └─ 快速理解价值 (Hero区域)
  │
  ├─ 问题2: "我能用吗？"
  │  └─ 展示简单易用 (演示 GIF / 视频)
  │
  ├─ 问题3: "值不值得用？"
  │  └─ 展示案例成果 + 用户评价
  │
  ├─ 问题4: "我现在就试试"
  │  └─ 快速注册 + Onboarding
  │
  ├─ 问题5: "我能完成吗？"
  │  └─ 第一个海报顺利生成 (关键转化点)
  │
  ├─ 问题6: "用得爽吗？"
  │  └─ 编辑体验流畅 + 结果满意
  │
  └─ 问题7: "付费值吗？"
     └─ Pro功能的性价比
```

### 减少摩擦点

```
【通常的摩擦点】        【我们的解决方案】

1. 不知道如何开始     ✓ 清晰的 Onboarding
                      ✓ 模板和示例

2. 操作步骤太多       ✓ 4-5 步完成首个设计
                      ✓ 智能推荐默认选择

3. 不知道如何调整     ✓ 所见即所得编辑器
                      ✓ 实时预览反馈
                      ✓ 在线帮助提示

4. 生成结果不满意     ✓ 自动生成多个变体
                      ✓ 一键快速调整
                      ✓ 细粒度编辑选项

5. 导出过程复杂       ✓ 一键导出
                      ✓ 预设常用格式
                      ✓ 批量导出支持

6. 效果如何评估       ✓ 预览多平台效果
                      ✓ 直接分享到社交
                      ✓ 使用统计反馈
```
