### 排版系统

```
H1 标题       48px / 700 Bold
             用于: 页面主标题

H2 副标题     32px / 600 SemiBold
             用于: 板块标题

H3 小标题     24px / 600 SemiBold
             用于: 卡片标题

Body 正文     14px / 400 Regular / 22px 行高
             用于: 主要内容

Caption       12px / 400 Regular
             用于: 辅助文本、时间戳
```

### 间距系统

```
间距递进: 4px → 8px → 12px → 16px → 24px → 32px → 48px

应用规则:
• 组件内间距: 8-16px (sm-lg)
• 组件外间距: 16-24px (lg-xl)
• 大块间距: 32-48px (xl-3xl)
```

### 圆角系统

```
4px   - 输入框、小按钮
6px   - 按钮、标签
8px   - 卡片、容器
12px  - 模态框
50%   - 完全圆形 (头像、徽章)
```

## 🎭 交互动画规范

### 过渡动画

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面加载 | Fade In | 300ms | ease-out |
| 模态打开 | Fade + Scale | 200ms | Cubic Bezier |
| 卡片悬停 | Elevation | 200ms | ease-out |
| 元素删除 | Fade + Slide | 200ms | ease-in |

### 微交互示例

```
【输入框焦点】
默认: border #CCCCCC
焦点: border #0066CC + 外发光
过渡: 200ms cubic-bezier

【按钮点击】
按下: scale(0.98)
松开: scale(1) with ripple effect

【拖拽反馈】
拖拽中: scale(1.05) + 阴影提升
释放: 自动吸附到网格

【加载动画】
进度条: 无限线性旋转 (1.5s周期)
骨架屏: 左右扫过 (2s周期)
```

```
  • 左导航 → 汉堡菜单
  • 右工具栏 → 底部 Sheet

  • 间距: 12px

Tablet (768-1199px):
  • 两列布局 (左侧 + 主区域)
  • 保留左侧导航
  • 右工具栏 → 底部 Sheet
  • 按钮高度: 40px
  • 间距: 16px

Desktop (1200px+):
  • 完整三列布局
  • 所有面板可见
  • 按钮高度: 36px
  • 间距: 24px
```

## 💡 设计决策记录

### 为什么选择这个配色方案？

```
蓝色 (#0066CC):
  + 传递专业感和信任
  + 科技行业标准色
  + 文案可读性高
  - 竞品很多也用蓝色

白色 + 浅灰:
  + 现代最小化美学
  + 清晰的信息层级
  + 打印友好
  + 无障碍对比度高
```

### 为什么选择三列编辑器布局？

```
优势:
  ✓ 充分利用宽屏空间
  ✓ 输入控件和预览并排
  ✓ 减少来回切换
  ✓ 高效的工作流程

折衷:
  ~ 窄屏需要 sheet 响应式
  ~ 对小显示器不友好

替代方案对比:
  单列布局     - 简单但需要标签页切换 ✗
  左右分栏     - 可行但输入框空间不足 ~
  三列最优     - 平衡性最好 ✓
```
